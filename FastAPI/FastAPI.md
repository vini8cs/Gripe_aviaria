Primeiramente instale P


O Poetry já cria uma estrutura automática para nosso preto

```bash
poetry new fast_zero
```

A estrututa fica:

```bash
├── fast_zero
│   └── __init__.py
├── pyproject.toml
├── README.md
└── tests
    └── __init__.py

```

Entre no diretório:

```bash
cd fast_zero
```

E agora vamos utilizar o py
